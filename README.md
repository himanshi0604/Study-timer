-Pomodoro Timer
A simple study timer web application built using HTML, CSS, and JavaScript.  
It helps users stay focused while studying using a 25-minute countdown timer inspired by the Pomodoro technique.

#Features
25-minute study timer
Start timer
Pause timer
Reset timer
Aesthetic background with animated stars
Minimal and clean UI

#Technologies Used
- HTML
- CSS
- JavaScript

-How It Works
1. Click Start to begin the 25-minute timer.
2. Click Pause to stop the timer temporarily.
3. Click Reset to restart the timer from the beginning.
